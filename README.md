# 깃허브 연습용 레포지토리

이 레포지토리는 **깃허브 사용 및 Git 실습**을 위해 만든 연습용 프로젝트입니다.  
커밋, 브랜치, PR(Pull Request), 머지 등의 기본적인 Git/GitHub 작업을 연습하는 용도로 사용됩니다.

---

## 프로젝트 목적
- Git과 GitHub 기본 명령어 실습
- 브랜치 생성, 커밋, 푸시, 풀, 병합(Merge) 연습
- README, 이슈, PR 작성 실습

---

## 사용 방법
1. 레포지토리 클론
```bash
git clone https://github.com/사용자이름/연습레포.git
cd 연습레포
```

2. 새로운 브랜치 생성 및 이동
```bash
git checkout -b feature/브랜치이름
```

3. 파일 수정 후 커밋
```bash
git add .
git commit -m "커밋 메시지"
```

4. 원격 저장소에 푸시
```bash
git push origin feature/브랜치이름
```

5. GitHub에서 Pull Request 생성 및 병합
